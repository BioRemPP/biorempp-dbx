## 1. Scope and Purpose

The **BioRemPP Database Explorer** is a scientific research tool designed for open, read-only exploration of the BioRemPP Integrated Database. The resource provides compound-centric access to curated associations between environmental compounds, degradation genes, metabolic pathways, toxicity predictions, and regulatory classifications.

BioRemPP Database Explorer is an **academic resource** provided on a best-effort basis without performance guarantees or service level agreements.

---

## 2. Intended Use

### Permitted Uses

- Academic research and hypothesis generation
- Educational purposes and training
- Exploratory analysis of bioremediation-related evidence
- Methodological development and validation
- Database browsing, filtering, and guided query exploration

### Prohibited Uses

!!! danger "Important Restriction"
    BioRemPP Database Explorer outputs must **NOT** be used as the sole basis for:

    - Clinical diagnostics or medical decisions
    - Regulatory submissions or compliance determinations
    - Environmental remediation decisions without independent validation
    - Commercial product claims without experimental confirmation

For complete scientific and methodological limitations, see [Assumptions and Limitations](../methods/assumptions-and-limitations.md).

## 4. Data Access and Privacy

The BioRemPP Database Explorer operates under a **privacy-by-design philosophy**:

- **No user accounts:** The resource requires no authentication or registration.
- **Read-only access:** No data submission, file upload, or user-generated content is involved.
- **Minimal technical logging:** Server logs contain only IP addresses (rate limiting), user-agent strings (compatibility), and error messages (debugging). No user data is stored or retained.
